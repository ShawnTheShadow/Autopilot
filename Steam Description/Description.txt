[b][u]Settings[/u][/b]
The file at "%AppData%\SpaceEngineers\Storage\ARMS\ServerSettings.txt" contains the settings for ARMS.
To reset a value to its default, simply delete it.
type "/arms settings" in game to see the current settings. Not all server settings will be displayed.
bAirResistanceBeta - enable air resistance for ships in atmosphere, does not work with Autopilot yet
bAllowAutopilot - this mod can control the movement of grids
bAllowGuidedMissile - allows guided missiles, requires bAllowWeaponControl
bAllowHacker - allows hacker block
bAllowRadar - radar can be used to detect grids, otherwise functions as a beacon
bAllowWeaponControl - Enables Weapon Control and E command
bImmortalMiner - prevents a mining Autopilot's ship from being damaged by an asteroid
bUseRemoteControl - set to true if you want to use remote controls as Autopilot blocks
yParallelPathfinder - Number of pathfinding tasks that can run in parallel
fDefaultSpeed - the desired speed, when not using V
fMaxSpeed - the maximum speed Autopilot is allowed to fly at
fMaxWeaponRange - the maximum range any weapon can be set to
